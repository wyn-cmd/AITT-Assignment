# AITT-Assignment
Log-Based Anomaly Detection for Cybersecurity (HDFS), a machine learning project focused on Behavioral Anomaly Detection within the Hadoop Distributed File System (HDFS). Developed for an AI Tool &amp; Techniques module.


Application utilises the HDFS datasets from Loghub.
